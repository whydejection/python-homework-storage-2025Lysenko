### Задание №1: Ввод 10 целых чисел с валидацией

Программа, которая создаёт список из 10 целых чисел, введённых пользователем.

**Требования:**
1. Запрашивать у пользователя ввод 10 целых чисел
2. Для каждого числа отображать его номер позиции
3. Обрабатывать ошибки ввода с использованием try/except
4. При ошибке ввода:
   - Выводить сообщение "Ошибка! Введите целое число"
   - Повторять запрос для той же позиции
5. После ввода всех 10 чисел выводить:
   - Список введённых чисел
   - Минимальное число
   - Максимальное число
   - Сумму всех чисел

**Пример использования:**
```
Программа для ввода 10 целых чисел
========================================
Введите целое число #1: 5
Введите целое число #2: abc
Ошибка! Введите целое число
Введите целое число #2: 10
Введите целое число #3: 15
...
```

**Файл:** `task01.py`
